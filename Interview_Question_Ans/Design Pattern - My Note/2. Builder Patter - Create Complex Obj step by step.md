### **ðŸ” Real Example: Building Burger**
```php
// 1. PRODUCT
class Burger {
    public function __construct(
        public $size,
        public $cheese = false,
        public $tomato = false
    ) {}
}

// 2. BUILDER
class BurgerBuilder {
    public function __construct($size) {
        $this->size = $size;
    }
    
    public function addCheese() {
        $this->cheese = true;
        return $this;
    }
    
    public function addTomato() {
        $this->tomato = true;
        return $this;
    }
    
    public function build(): Burger {
        return new Burger($this->size, $this->cheese, $this->tomato);
    }
}

// 3. USAGE
$burger = (new BurgerBuilder('Large'))
    ->addCheese()
    ->addTomato()
    ->build();
```
